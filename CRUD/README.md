## 路由设计

| 请求方法 | 请求路径 | get参数 | post参数 |       备注       |
| :------: | :------: | :-----: | :------: | :--------------: |
|   GET    |    /     |         |          |     渲染首页     |
|   GET    |   /new   |         |          | 渲染添加学生页面 |
|   POST   |   /new   |         |   yes    | 处理添加学生请求 |
|   GET    |  /edit   |   yes   |          |   渲染编辑页面   |
|   POST   |  /edit   |         |   yes    |   处理编辑请求   |
|   GET    |   /del   |   yes   |          |   处理删除请求   |
